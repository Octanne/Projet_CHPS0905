IMAGE SELECTION


IMAGES are displayed on screen if you double left click on a SERIES displayed in the TREE VIEW.

Selection of an IMAGE is done by a left click. A selected IMAGE will appear with a green frame around it. Selecting a new image by doing a left click on it, will unselect the previous selected one.
Selection or unselection of several IMAGES is done by doing successive left clicks on different IMAGES while keeping the Ctrl key pressed on the keyboard. 

If synchronization of series is not activated, zooming, windowing, rotations, flips and JPEG export will only apply on the selected image(s).
