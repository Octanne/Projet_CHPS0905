DICOM TREE VIEW


Dicom files are represented as a DICOM tree. Each line has a level in the tree and gives an information or a group of informations on the level.

There are 3 levels in the tree view: PATIENT, STUDY and SERIES.

• PATIENT level.
It gives identity and sex of the PATIENT.

• STUDIES level. 
Every other line represents a STUDY. The date of the exam is mentioned.

• SERIES level.
If it exists, the first line represents the SYNTHESIC LIST of key images. A left double click on this line will display these images.
If it exists, the second line represents the REPORT linked to the STUDY. A left click on this line will display the DICOM Structured Report in an additional window.
Every other line represents a SERIES, giving modality type (CT for scanner, MR for magnetic resonnance, etc) and the number of images in the SERIES.
