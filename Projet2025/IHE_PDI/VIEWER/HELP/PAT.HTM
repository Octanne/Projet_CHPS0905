PATIENT FILE

The PATIENT FILE is organised according to the tree structure defined in the DICOM standard :

• The PATIENT FILE contains one or several STUDIES

• Each STUDY contains one or several SERIES

• Each SERIES contains one or several IMAGES


In addition, a SYNTHESIS FILE composed of significant IMAGES from the STUDIES can be present. It groups some key images from the SERIES of the whole PATIENT FILE.

If it has been included, you can also find a text REPORT for the STUDY.
