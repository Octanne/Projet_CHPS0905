DISPLAY GSPS


This TOOL is available using button DEFAULT PRESENTATION . 

Applying a GSPS object to the raw image will exactly reproduce the presentation of the image (windowing, zooming, panning, rotations, measurements, annotations) made by the technologist or the radiologist.

After clicking on the drop list, you can apply  :
- the default presentation (without GSPS object applied)
- the Grayscale Softcopy Presentation State object
